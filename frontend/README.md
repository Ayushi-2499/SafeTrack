# 🚀 SafeTrack

SafeTrack is an IoT-enabled smart safety and intrusion detection system developed using MERN stack and Raspberry Pi.

## 🔥 Features

- Real-time intrusion detection
- Sensor-based alert system
- Live frontend dashboard
- Backend API integration
- IoT device connectivity
- Alert notifications using buzzer and LED

---

## 🛠 Technologies Used

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

### IoT Hardware
- Raspberry Pi
- IR Sensor
- LED
- Buzzer
- Servo Motor

## 🚀 How to Run

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
node server.js
```
